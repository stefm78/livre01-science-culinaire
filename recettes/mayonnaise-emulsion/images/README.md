## Instructions de génération

1. Utiliser un générateur d'images IA (Midjourney, DALL-E 3, Stable Diffusion)
2. Copier les prompts ci-dessus exactement
3. Sélectionner les meilleures générations (haute résolution, réalisme)
4. Nommer les fichiers selon la convention : `hero.png`, `final.png`, `process-01.png`, `process-02.png`
5. Optimiser les images (compression sans perte, dimensions 1920x1080 pour hero/final, 1200x800 pour process)
6. Placer les fichiers dans le répertoire `recettes/mayonnaise-emulsion/images/`
