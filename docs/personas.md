# Personas du Projet

Ce document contient la documentation des personas décrits dans la Constitution projet IA, pour référence et compréhension du système d'orchestration multi-IA.

## Personas Standardisés

Chaque persona correspond à un rôle spécialisé dans le workflow projet :

- **chef-de-projet-ia.md** : Coordination et cadrage
- **analyste-stratege.md** : Analyse stratégique et planification
- **executant-specialiste.md** : Réalisation technique
- **researcher-veilleur.md** : Recherche et veille
- **creatif-designer.md** : Création et design
- **analyste-donnees.md** : Analyse de données
- **communicant-redacteur.md** : Communication et rédaction
- **controleur-qualite.md** : Contrôle qualité

## Structure de Persona

Chaque fichier persona inclut :
* Les responsabilités spécifiques
* Les compétences requises
* Les circonstances de mobilisation
* Les étapes du workflow associées

## Usage

Les personas sont mobilisés via les labels `persona:nom-persona` dans les issues, conformément à la Constitution du projet et aux workflows d'orchestration IA.
